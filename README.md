# 🎓 AI Career Guide

An AI-powered Career Guidance platform built using Python, Flask, HTML, CSS and Gemini AI.

This project helps students explore career opportunities, colleges, scholarships, AI guidance, resume building, and personalized career roadmaps.

---

## 🚀 Features

### ✅ AI Career Recommendations
- Career suggestions based on selected stream.
- Supports PCM, PCB, Commerce and Arts.

### ✅ KCET College Suggestions
- College recommendations based on KCET Rank.

### ✅ Scholarship Suggestions
- Displays scholarships based on education level.

### ✅ Gemini AI Career Chatbot
- Ask career-related questions using Gemini AI.

### ✅ AI Resume Builder
- Professional Resume Generator.

### ✅ Resume PDF Download
- Download resume as PDF.

### ✅ AI Career Roadmap Generator
- Step-by-step roadmap for chosen career.

### ✅ Modern Premium UI
- Cyber-style dark interface.
- Professional animations.
- Transparent cards.
- Background graphics.
- Responsive layout.
